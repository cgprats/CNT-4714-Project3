NOTE: These Commands Assume java and javac are in your PATH
To Compile the Project, Run the Following Command:
Mac and Linux: javac -cp .:mysql-connector-java-8.0.22.jar SQLClient.java QueryResultTable.java
Windows: javac -cp '.;mysql-connector-java-8.0.22.jar' SQLClient.java QueryResultTable.java
To Execute the Project, Run the Following Command:
Mac and Linux: java -cp .:mysql-connector-java-8.0.22.jar SQLClient
Windows: java -cp '.;mysql-connector-java-8.0.22.jar' SQLClient
